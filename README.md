## Getting Started

### Clone Code from ciresan and theanet

`git submodule update --init --recursive`

### Install Lasange

```
git clone https://github.com/Lasagne/Lasagne.git
cd Lasagne/
pip install -r requirements.txt
sudo python setup.py install
```
